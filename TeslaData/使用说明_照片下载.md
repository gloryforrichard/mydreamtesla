# Tesla 照片下载器 — 使用说明

## 前提条件

1. **Python 3.8+**（已安装）
2. **Google Chrome 浏览器**（已安装在您电脑上）
3. 联网环境

---

## 第一步：安装依赖

打开终端（Mac/Linux）或命令提示符（Windows），在本文件夹下运行：

```bash
pip install -r requirements_tesla_downloader.txt
```

---

## 第二步：运行脚本

```bash
python tesla_photo_downloader.py
```

脚本会自动：
- 打开无头Chrome浏览器访问各图库网站
- 滚动页面触发懒加载，提取所有高清图片URL
- 按型号/世代/内外分类下载到 `Tesla_Photos/` 文件夹
- 完成后自动生成 HTML 浏览索引

---

## 预计时间

- 约 25–45 分钟（取决于网速和网站响应）
- 约 500–1500 张照片（视各网站实际图库大小）
- 约 500MB–2GB 磁盘空间

## 浏览照片

下载完成后，**双击打开 `Tesla_Photos/📸 Tesla照片库_双击打开.html`**

功能包括：
- 按型号/世代/内外分类展示缩略图
- 点击任意图片放大查看（灯箱效果）
- 顶部导航快速跳转
- 完全离线可用

## 常见问题

**Q: 脚本报错 "无法启动Chrome"**
A: 确保已安装 Google Chrome，`webdriver-manager` 会自动下载对应版本的ChromeDriver。

**Q: 某个网站图片数量很少**
A: 部分网站有反爬措施，可能只抓到部分图片。脚本已做了人类行为模拟（滚动、延迟），但无法100%保证。

**Q: 下载速度很慢**
A: 脚本默认每张图片间隔0.5秒（`DOWNLOAD_DELAY`），可在脚本顶部将其改小，但过快可能触发限流。

**Q: 想重新下载**
A: 删除 `Tesla_Photos/` 文件夹后重新运行即可。脚本会自动跳过已存在的文件。
