## Tesla Model 3 北美版核心规格数据库 (2017–2026)

> **📌 北美口径与数据基准说明**
> * **适用范围**：本文档仅覆盖美国与加拿大（US/Canada）交付版本。
> * **核心数据来源**：续航采用 EPA 预估值（美国以 mi 显示，加拿大官网以 km 显示）；车身尺寸与货厢容积优先采用 Tesla 北美官方车主手册（Owner's Manual）。
> * **非公开数据标注**：Tesla 在北美官网及手册中并不系统公开老款车型的电池总容量及精确马力/扭矩。表格中涉及此类参数的列已明确标注为“社区常见推算值”或“部分官方宣传口径”，实际表现可能因年款、工厂批次与 OTA 软件版本而存在差异。
> * **命名说明（Highland 段）**：2024+ 北美官网命名在不同时间/地区页面可能出现 RWD / Premium RWD / Premium AWD 与 Long Range 命名并存或映射关系，本文在表格中以当前北美官网常见命名展示，并提供 US/CA 对照。

### 🧠 核心硬件防坑时间线（北美交付特征）

* **车机系统 (MCU)**：Intel Atom (2017 - 2021) ➔ AMD Ryzen（约 2022 年批次起过渡）。
* **低压电瓶**：12V 铅酸电池 (2017 - 2021) ➔ 16V 锂电池（约 2022 年批次起过渡）。
* **毫米波雷达**：北美版 Model 3 在 2021 年前后进入 Tesla Vision（纯视觉）过渡，逐步取消毫米波雷达。
* **超声波雷达 (USS)**：2022 年起新生产车辆分批取消（多数市场先行），2023 年后北美新车基本以纯视觉方案为主。
* **补充说明**：上述硬件时间线以 Tesla Support 公告与北美交付观察为参考，实际仍以车辆生产日期、VIN 批次与地区配置为准。
